# Отклик на вакансию через AdsPower

## Что делает этот скилл

Открывает вакансию на hh.ru в браузере AdsPower, помогает откликнуться: вставляет сопроводительное письмо, прикрепляет CV. Всегда ждёт подтверждения пользователя перед отправкой.

## Как использовать

`/apply 12345678` (ID вакансии) или `/apply https://hh.ru/vacancy/12345678`

## Инструкции

### Шаг 1: Подготовка

1. Прочитай `data/tracker.md` — найди вакансию по HH ID
2. Если вакансии нет в трекере — сначала предложи оценить: `/evaluate {ID}`
3. Найди отчёт оценки в `reports/` — возьми черновик сопроводительного из блока "Сопроводительное письмо"
4. Найди сгенерированный PDF в `output/` (если есть)

### Шаг 2: Открытие браузера

1. Получи список профилей AdsPower:
   - Используй `get-browser-list` чтобы найти нужный профиль
   - Если профилей несколько — спроси пользователя какой использовать

2. Открой профиль:
   - Используй `open-browser` с profileId
   - Получи ws URL из ответа

3. Подключись к браузеру:
   - Используй `connect-browser-with-ws` с полученным wsUrl

### Шаг 3: Переход на вакансию

1. Используй `navigate` для перехода на `https://hh.ru/vacancy/{ID}`
2. Подожди загрузку страницы
3. Сделай скриншот для пользователя — `screenshot`

### Шаг 4: Проверка авторизации

1. Проверь `get-page-visible-text` — есть ли кнопка "Откликнуться" или "Войти"
2. Если нужна авторизация:
   - Сообщи пользователю: "Нужно авторизоваться на hh.ru. Войди в аккаунт и скажи когда готово."
   - Дождись подтверждения
   - Сделай скриншот для проверки

### Шаг 5: Отклик

1. Найди кнопку "Откликнуться" на странице
2. **СТОП — покажи пользователю:**
   - Название вакансии и компанию
   - Оценку из трекера
   - Черновик сопроводительного письма
   - Есть ли готовый PDF
   - **Спроси: "Откликаемся? Хочешь изменить сопроводительное?"**

3. После подтверждения:
   - Нажми кнопку "Откликнуться" (`click-element`)
   - Если появилась форма с полем для сопроводительного:
     - Вставь текст (`fill-input`)
   - Если есть возможность прикрепить файл — сообщи пользователю путь к PDF
   - Сделай скриншот перед финальной отправкой
   - **Спроси ещё раз: "Всё верно? Отправляем?"**

4. После отправки:
   - Сделай скриншот подтверждения
   - Обнови статус в трекере на "Отклик"

### Шаг 6: Завершение

Сообщи пользователю:
- Отклик отправлен на {Компания} — {Должность}
- Трекер обновлён: статус = Отклик
- Предложи: "Откликнуться на следующую вакансию?"

## Важные правила

1. **НИКОГДА не отправляй отклик без подтверждения пользователя** — это главное правило
2. Всегда показывай скриншот перед отправкой
3. Если что-то пошло не так (кнопка не найдена, ошибка) — сообщи и не пытайся обойти
4. Если нет сопроводительного письма — предложи сгенерировать через /evaluate
5. Если нет PDF — предложи сгенерировать через /cv-gen

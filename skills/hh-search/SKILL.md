# Поиск вакансий на hh.ru

## Что делает этот скилл

Ищет вакансии через API hh.ru по заданным запросам из `templates/queries.yml`, фильтрует по ключевым словам, дедуплицирует с трекером, выводит список новых вакансий.

## Как использовать

Пользователь вызывает: `/hh-search` или `/hh-search "свой запрос"`

## Инструкции

1. Если пользователь указал конкретный запрос — запусти:
   ```bash
   node ${CLAUDE_PLUGIN_ROOT}/scripts/search-hh.mjs --query "запрос пользователя" --limit 10
   ```

2. Если без аргументов — запусти все запросы:
   ```bash
   node ${CLAUDE_PLUGIN_ROOT}/scripts/search-hh.mjs --all
   ```

3. Прочитай вывод JSON из stdout (или из `data/last-search.json`).

4. Выведи результаты пользователю в виде таблицы:

   | # | Компания | Роль | ЗП | Формат | Ссылка |
   |---|----------|------|----|--------|--------|

5. Для каждой вакансии покажи краткое описание (snippet).

6. Спроси пользователя: какие вакансии оценить подробнее? (можно указать номера)

7. Если пользователь выбрал вакансии — для каждой вызови скилл `evaluate` с ID вакансии.

## Заметки
- API hh.ru не требует авторизации для поиска
- Результаты сохраняются в `data/last-search.json`
- Дедупликация идёт по HH ID из `data/tracker.md`
- Если API не отвечает — предложи проверить подключение к интернету
- Скрипт `search-hh.mjs` читает конфиг из `templates/queries.yml`. Если пользователь работает в другой директории, нужно убедиться что queries.yml доступен относительно рабочей папки или указать путь явно.
